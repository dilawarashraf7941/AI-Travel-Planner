# AI Travel Planner using LangGraph

An AI-powered Travel Planner built with Python, LangGraph, and Google Gemini.

## Features

- Multi-Agent Workflow
- LangGraph StateGraph
- Planner Agent
- Travel Information
- Hotel Recommendations
- Flight Information
- Activity Recommendations
- AI Itinerary Generator
- Error Handling
- Fallback Itinerary

## Tech Stack

- Python
- LangGraph
- Google Gemini API
- dotenv

## Project Structure

```
Travel_Planner/
│
├── agents/
├── graph/
├── tools/
├── config.py
├── utils.py
├── main.py
├── requirements.txt
└── README.md
```

## How to Run

```bash
pip install -r requirements.txt

python main.py
```
